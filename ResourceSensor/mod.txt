A new automation sensor that outputs logic depending on selected materials, amount and count mode. Building is unlocked after Generic Sensors research and uses 25Kg of Refined metal. 

There are 3 selectable counting modes:

[list]
[*][b]Distance Mode[/b]
Count materials around the sensor up until specified distance.
[*][b]Room Mode[/b]
Count materials which are in the same room as the sensor.
[*][b]Global Mode[/b]
Count materials that are located in the current active cluster. This mode only counts dupe-accessible resources.
[list]
